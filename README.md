# Receipt Scanner App

A React Native Expo app for scanning receipts and tracking expenses.

## Features

- 📷 Camera scanning
- 🖼️ Gallery selection
- 🤖 AI-powered parsing
- 💾 Local storage
- 📊 Expense charts
- 🏷️ Category tracking

## Setup

1. Clone repository
2. Run `npm install`
3. Add Gemini API key in `utils/api.js`
4. Run `expo start`

## API Keys

- Gemini API: Get from Google AI Studio
- OCR.space: Free tier included
